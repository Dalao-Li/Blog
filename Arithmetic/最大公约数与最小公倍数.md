# 欧几里得算法

最大公约数 gcd

```c
ll gcd(ll a, ll b){
    return b == 0 ? a :gcd(b, a%b);
}
```

> C++ algorithm 库内置有`__gcd(a,b)`函数用以求最大共约数

最小公倍数 lcm

lcm(a,b) = $\frac{a*b}{gcd(a,b)}$

```c
//先除再乘避免溢出
ll lcm(ll a, ll b){
    return a/gcd(a,b) * m;
}
```
