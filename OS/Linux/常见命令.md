# 更新

- apt-get update

> 同步 /etc/apt/sources.list 和 /etc/apt/sources.list.d 中列出软件源索引

- apt-get upgrade

> 对系统执行更新

# 清理内存

```shell
echo 1 > /proc/sys/vm/drop_caches
```

- 0 : 0 是系统默认值,默认情况下表示不释放内存,由操作系统自动管理

- 1 : 释放页缓存

- 2 : 释放 dentries 和 inodes

- 3 : 释放所有缓存
