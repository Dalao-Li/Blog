## 查看远程仓库

```shell
git remote -v
```

## 删除远程仓库

```shell
git remote rm [origin name]
```

## 添加新地址

```shell
git remote add [origin name] [origin url]
```

## 添加多个远程地址

在使用git时,可将库同时push到多个托管远端,如防止git托管不稳定,起备份作用

```sh
git remote set-url --add origin [URL]
```
假设想要增加2个远程库地址,分别为
```sh
https://github.com/[仓库]

https://code.aliyun.com/[仓库]
```

解决方法
```sh
# 添加首个远程地址
git remote add origin https://github.com/[仓库]


# 添加第二个地址
git remote set-url --add origin https://code.aliyun.com/[仓库]
```

![](https://cdn.hurra.ltd/img/20210125231032.png)

此时`git push origin master` 就可一次性push到多个库里
