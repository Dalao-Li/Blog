## for循环

- 累计循环
```sh
#!/bin/bash

for i in {1..5}

do
    echo $i;
done
```

![](https://cdn.hurra.ltd/img/20210310152256.png)

- 遍历

```sh
#!/bin/bash

for i in This is a string

do
    echo $i;
done
```

## while循环

```sh
#!/bin/bash

i=1

while(( $i<=5 ))
do
    echo $i
    let "int++"
done
```
