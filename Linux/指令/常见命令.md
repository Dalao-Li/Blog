## 更新源

```sh
# 同步sources.list 和sources.list.d 中列出软件源索引

apt-get update
```

```sh
# 对系统执行更新
apt-get upgrade
```

## 清理内存

```sh
echo [参数] > /proc/sys/vm/drop_caches
```
| 参数 | 含义                                                   |
| ---- | ------------------------------------------------------ |
| 0    | 系统默认值,默认情况下表示不释放内存,由操作系统自动管理 |
| 1    | 释放页缓存                                             |
| 2    | 释放 dentries 和 inodes                                |
| 3    | 释放所有缓存                                           |
