## 单行追加

```sh
echo "[文本内容]" >> [文件名]
```

追加内容

```sh
echo "追加内容1" >> demo

echo "追加内容2" >> demo
```

![](https://cdn.hurra.ltd/img/20200619212430.png)

## 多行追加

```sh
echo -e "[第一行内容]\n[第二行内容]" >> [文件名]
```

## 内容覆盖

```sh
echo "[文本内容]" > [文件名]
```
