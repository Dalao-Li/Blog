新建文件demo,原本内容为空
```
touch demo
```

## 1.1 内容追加
```
echo "文本内容" >> 文件名
```

追加内容
```
echo "追加内容1" >> demo

echo "追加内容2" >> demo
```

## 1.2 内容覆盖
```
echo "文本内容" > 文件名
```

覆盖内容
```
echo "覆盖内容" > demo
```
