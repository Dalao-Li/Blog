# 一、目的
通过echo命令实现对文件的内容进行追加与覆盖

# 二、环境
WSL(Ubuntu18.04)

# 三、过程


新建文件demo,原本内容为空
```
touch demo
```

## 3.1 内容追加
```
echo "文本内容" >> 文件名
```

追加内容
```
echo "追加内容1" >> demo

echo "追加内容2" >> demo
```
![](http://cdn.hurra.ltd/img/20200619212430.png)

## 3.2 内容覆盖
```
echo "文本内容" > 文件名
```

覆盖内容
```
echo "覆盖内容" > demo
```
