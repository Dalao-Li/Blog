## 匹配模式

```sh
sed -[参数] '[启示范围,结束范围]  [操作]' [文件]

# 例如
sed -n '1,5 p' file.txt
```

| 参数           | 含义                                                     |
| -------------- | -------------------------------------------------------- |
| n              | 忽略执行过程的输出只输出结果                             |
| i  `非常危险!` | 所有改动将覆盖原文件 ; 最好换为`-i.bak` 对原文件生成备份 |

| 范围取值      | 含义                                        |
| ------------- | ------------------------------------------- |
| 5             | 选择第5行,若为空则是选择所有行              |
| 2,5 /或 2,+3  | 选择2到5行,共4行                            |
| 1~2           | 选择奇数行                                  |
| 2~2           | 选择偶数行                                  |
| 2,$           | 从第2行到文件结尾                           |
| /sys/,+3      | 选择出现sys字样的行,以及后面的三行          |
| /\^sys/,/mem/ | 选择以sys开头的行,和出现mem字样行之间的数据 |

| 操作 | 含义                                |
| ---- | ----------------------------------- |
| p    | 对匹配内容进行打印                  |
| d    | 对匹配内容进行删除,此时要去掉-n参数 |
| w    | 将匹配内容写入其他地方              |
