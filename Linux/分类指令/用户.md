## 查看用户

```sh
cat /etc/passwd
```

- 格式化输出

```sh
cat /etc/passwd|grep -v nologin|grep -v halt|grep -v shutdown|awk -F":" '{ print $1"|"$3"|"$4 }'|more
```

## 查看组

```sh
cat /etc/group
```


## 新建用户

```sh
sudo adduser [用户名]

sudo passwd [用户名]
```
