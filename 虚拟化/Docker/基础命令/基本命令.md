## 删除所有容器

```docker
docker rm $(docker ps -aq)
```

## 停用并删除容器

```docker
docker stop $(docker ps -q) & docker rm $(docker ps -aq)
```

## 文件复制

- 宿主机->容器
```sh
docker cp <宿主机目录/文件> <容器id>:<容器内目录/文件>
```

- 容器->宿主机
```sh
docker cp <容器id>:<容器内目录/文件> <宿主机目录/文件>
```
