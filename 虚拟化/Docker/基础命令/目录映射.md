# Docker目录挂载


- 指令

> docker -v <宿主机目录>:<容器目录>

## 注意

- 容器目录只能为绝对路径

> 绝对路径,必须以下斜线“/”开头

将宿主机 当前目录下的/myapp目录映射到容器内/usr/src/myapp 目录

![](https://cdn.hurra.ltd/img/20210109220453.png)

- 宿主机目录若不存在会自动生成

将宿主机 demo 目录(不存在) 映射到容器内/demo 目录

![](https://cdn.hurra.ltd/img/20210109223115.png)
