## 函数模板

求和函数,返回两个输入值的和

- 函数定义
```c
template <class T>
T add(T a,T b) {
	return a + b;
}
```

测试代码

```c
#include <iostream>
using namespace std;

template <class T>
T add(T a,T b) {
	return a + b;
}

int main() {
	cout<<add<int>(1,2)<<endl;
	cout<<add(2.4,3.5)<<endl;
	cout<<add<string>("I am ","a student.")<<endl;
	return 0;
}
```
