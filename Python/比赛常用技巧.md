# 输入输出

## 多输入隔行输出

- 第一行给定两个值 n,m，用空格分割，第一个 n 决定接下来有 n 行的输入，m 决定每一行有多少个数字，m 个数字均用空格分隔.

```py
# 接收两个值，使用n,m分别接收列表中的两个值
n, m  = [int(i) for i in input().split()]

# 构造输入列表的列表
num_list = []

for i in range(n):
	# 不用在意m的值，将所有数值接收进来然后用len判断长度
	tmp_list = [int(i) for i in input().split()]
	num_list.append(tmp_list)
```
