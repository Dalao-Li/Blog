# 三、过程

## 3.1 Pytohn

```python
name = "Dalao"
age = 21
```

1. 方法 1

```py
info = "My name is %s , I am %d years old."%(name,age)
```

2. 方法 2 format

```py
info = "My name is {0} , I am {1} years old.".format(name,age)
```

3. 方法 3 字典传值

```py
data = {'name':"Dalao",'age':21}

info = "My name is %(name)s , I am %(age)d years old."%data
```
