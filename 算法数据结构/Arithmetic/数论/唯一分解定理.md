# 算术基本定理

又称为正整数的唯一分解定理,即:每个大于1的自然数,要么本身就是质数,要么可以写为2个或以上的质数的积,且这些质因子排列后,写法仅有一种方式

> 即 $N =p_1^{a_1}p_2^{a_2}...p_k^{a_k} = \prod_{i=1}^k p_i^{a_i};其中p_1,p_2...p_k<N且均为质数$

- 例如: $6936=2^3*3*17^2,1200=2^4*3*5^2$

## 分析  
对于$N =p_1^{a_1}p_2^{a_2}...p_k^{a_k}$,其因数个数为:$(a_1+1)*(a_2+1)*...*(a_k+1)$

## 证明:  
$N =p_1^{a_1}p_2^{a_2}...p_k^{a_k}$,其因数包括:
- $包含0个p_1时,x_1 = p_2^{a_2}...p_k^{a_k}$
- $包含1个p_1时,x_2 = p_1^{1}p_2^{a_2}...p_k^{a_k}$  
- $.......$
- $包含a_1个p_1时,x_{a_1} = p_1^{a_1}p_2^{a_2}...p_k^{a_k}$

即对于$p_1$部分来说,可得到N的因数个数为$(a_1+1)个$,所以N的因数个数为:$(a_1+1)*(a_2+1)*...*(a_k+1)$

例: $24=2^3*3^1$ 则24的因数个数为:(3+1)*(1+1) = 8

## 解题
```c
#include<iostream>
#include<map>
typedef long long ll;
using namespace std;

int main() {
    map<int, int> m;
    for (int i = 1; i <= 100; i++) {
        int x = i;
        for (int j = 2; j * j <= x; j++) {
            //当j是x的素因子时
            while (x % j == 0) {
                //m[j]表示素因子j的指数
                m[j]++;
                x /= j;
            }
        }
        //若x不为1说明其为素数
        if (x != 1) {
            m[x]++;
        }
    }
    ll sum = 1;
    for (auto &it : m) {
        sum *= (it.second + 1);
    }
    cout << sum;
}
```
