# 一、提出
求 $3^{13}$

# 二、研究

## 2.1 常规算法
$3^{13}$ = 3 * 3 * 3 * 3 * 3 * 3 * 3 * 3 * 3 * 3 * 3 * 3 * 3 
需要算14次,太麻烦,不符合程序员的逼格

## 2.2 快速幂  
$(13)_{10}$ = $(1101)_2$ = 1 * $2^3$ +  1 * $2^2$ + 0 * $2^1$ + 1 * $2^0$

$3^{13}$ = $3^{1 * 2^3 +1 * 2^2 + 0 * 2^1 + 1 * 2^0
}$ 

$3^{13}$ = $3^8$ * $3^4$ * $3^1$

需要算7次,方便运算

推导过程
$3^1$  =  $(3^1)$  

$3^2$  =  $(3^1)^2$ 

$3^4$ =  $(3^2)^2$  

$3^8$ = $(3^4)^2$  

>快速幂就是运算过程不断对底数进行平方运算，并当指数的二进制某位为 1时,保存当前底数的值
## 2.3 代码实现
代码实现
```c
typedef long long ll;
//a为底数,b为指数
ll fastPower(int a,int b) {
	if(b==0) {
		return 1;
	}
	//res用来保存结果
	ll res = 1;
	while(b) {
		//指数当前位为1时,保存底数的值
		if(b&1) {
			res*=a;
		}
		//判断指数的下一位
		b>>=1;
		//底数做平方运算
		a*=a;
	}
	return res;

}
```
