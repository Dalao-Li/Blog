> 运行环境:CentOS7

## 搭建JDK

```sh
# 安装jdk1.8
yum -y install java-1.8.0-openjdk*

# 查看java版本
java --version
```

## 安装Jenkins

下载Jenkins

```sh
wget http://mirrors.jenkins.io/war-stable/latest/jenkins.war
```

运行Jenkins

```java
java -jar jenkins.war --httpPort=8080
```
